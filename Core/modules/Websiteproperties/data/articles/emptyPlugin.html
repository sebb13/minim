Module vierge

Pour vous faciliter la tâche, ce module vide va vous fournir un squelette type sur lequel vous allez pouvoir ranger vos fichiers.

Codez tranquillement vos spécificités, applications, traitements, en utilisant tous les outils que minim met à votre disposition pour vous simplifier le travail.

En utilisant les fonctionnalités natives de minim, vous assurez également des temps de réponse très rapides.

Installation

Comme pour tous les modules, n'oubliez pas de :

  • Copier les classes PHP (présentes dans /lib) dans le répertoire Core/lib/Modules/ de votre site/application.
  • Ajouter les fichiers css et js dans votre configuration (/Core/data/concat_js.xml et /Core/data/concat_css.xml) afin qu'ils soient pris en compte.
  • Régénérer le cache afin que le module soit pris en compte.
