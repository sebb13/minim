Galleries

Le module Galleries permet de gérer des albums photos.

Il dépend du module Media pour fonctionner.

Installation

Tout comme le module media, ce module nécessite une base de données.

Vous trouverez les requêtes SQL d'installation dans le répertoire data du module.

Et comme pour tous les modules, n'oubliez pas de :

  • Copier les classes PHP (présentes dans /lib) dans le répertoire Core/lib/Modules/ de votre site/application.
  • Ajouter les fichiers css et js dans votre configuration (/Core/data/concat_js.xml et /Core/data/concat_css.xml) afin qu'ils soient pris en compte.
  • Régénérer le cache afin que le module soit pris en compte.

L'accueil de ce plugin vous permet de :

  • Gérer les galeries actives.
  • Gérer les galeries archivées.
  • Ajouter une nouvelle galerie.
  • Visualiser les galeries actives et y accéder.
  • Gérer la configuration du module.

Dashboard

Comme tous les modules (bien que ce ne soit pas indispensable), le module de gestion de galeries propose un widget sur l'accueil du backoffice avec un accès rapide aux galeries.

Accès aux galeries

A partir de cette interface, vous pouvez :

  • Gérer les galeries actives.
  • Gérer les galeries archivées.
  • Archiver une galerie.

Vous retrouvez la pochette et la description de vos galeries en guise menu, il vous suffit ensuite de cliquer sur la galerie que vous souhaitez éditer pour accéder à son interface de gestion.

Gestionnaire de galerie

A partir de cette interface, vous pouvez :

  • Accéder aux galeries actives et aux galeries archivées.
  • Archiver la galerie.
  • Ajouter un ou des média(s) à la galerie.
  • Éditer un média (accès direct au média sur le gestionnaire de média).
  • Supprimer un média de la galerie.
  • Supprimer un média de la galerie et de vos médias.
