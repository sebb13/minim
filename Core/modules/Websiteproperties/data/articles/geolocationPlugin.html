Géolocalisation

Pour bon nombre d'activités ou simplement pouvoir afficher son point de vente sur une carte, la géolocalisation est devenue indispensable.

Minim vous propose ici un module simple permettant de vous localiser ainsi que les visiteurs de votre site/application.

Ce module offre également une API vous permettant de faciliter la vie de vos utilisateurs dans la saisie d'adresse en s'appuyant sur une API gouvernementale afin de bénéficier de données à jour.

Des modules de gestion de carte vont venir bientôt afin de compléter ce module.

Installation

Comme pour tous les modules, n'oubliez pas de :

  • Copier les classes PHP (présentes dans /lib) dans le répertoire Core/lib/Modules/ de votre site/application.
  • Ajouter les fichiers js dans votre configuration (/Core/data/concat_js.xml) afin qu'ils soient pris en compte.
  • Ajouter au menu du backoffice la page d'accueil du module dans le menu "modules".
  • Régénérer le cache afin que le module soit pris en compte.

Configuration

  • FORMAT: Le format des données renvoyées par l'API
  • LATITUDE: votre latitude
  • LONGITUDE: votre longitude
  • RADIUS: le rayon autour de votre position à afficher sur la carte (en prévision des modules de gestion de carte).

Le bouton "localisez-moi" rempli automatiquement les champs latitude et longitude en fonction de votre position actuelle.

API

Comme dit plus haut, ce module offre une API reposant sur des données du gouvernement. Cette fonctionnalité n’est donc disponible pour l'instant que pour la France.

Voici les fonctionnalités offertes par l'API :

  • https://domain.ltd/FR/geoloc/getPositionFromAddress.html?address=paris
    Attend une valeur "address" en argument
    Renvoie un tableau avec les coordonnées trouvées.
  • https://domain.ltd/FR/geoloc/getAddressFromPosition.html?latitude=43.454029&longitude=5.475256
    Attend les valeurs "latitude" et "latitude" en arguments
    Renvoie un tableau avec les lieux trouvés.
  • https://domain.ltd/FR/geoloc/getAddress.html?address=gardanne&latitude=43.454029&longitude=5.475256
    Attend une valeur "address" en argument et optionnellement les valeurs "latitude" et "longitude" qui auront la priorité dans le classement des résultats.
    Renvoie un tableau avec les lieux trouvés.
  • https://domain.ltd/FR/geoloc/getPostCodeFromCity.html?city=gardanne
    Attend une valeur "city" en argument
    Renvoie un tableau avec les codes postaux trouvés.
  • https://domain.ltd/FR/geoloc/getCitiesFromPostCode.html?postcode=13790
    Attend une valeur "postcode" en argument
    Renvoie un tableau avec les villes trouvées.

Format

Le format de sortie de cette API se défini dans la configuration du module (json, XML ou un simple tableau PHP pour une utilisation interne au framework).
